Simplest OS ever.
Also useless, but the idea is to get a basic aproach of how to start on OS development.


Dependencies
------------

* nasm
* gcc
* ld
* qemu
* make


Compile & Run
-------------

First you need to compile the kernel with `make`, now you are ready to run it with `make run`
