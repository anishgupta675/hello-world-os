hello-world-os
==============

Un kernel muy basico, simple e inutil que sirve de base para entender como empezar a desarrollar un SO.


Dependencias
------------

* nasm
* gcc
* ld
* qemu
* make


Compilar & Correr
-----------------


Primero tenes que compilar el  kernel con `make`, ahora deberias poder correrlo con `make run`
